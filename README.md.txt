# Proiect Sincretic - Arduino + Flask

## Descriere

Acest proiect implementează un sistem cu Arduino, senzor de temperatură, senzor de apă, LED și interfață web Flask.

## Componente

- Arduino Uno/Nano
- Senzor LM35
- Senzor apă
- LED + rezistor
- Breadboard

## Rulare aplicație Flask

```bash
pip install -r requirements.txt
python app.py
